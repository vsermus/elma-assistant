# Отображение данных пользователей

**Правило:** если в дашборде, отчёте или ответе AI-агента выводятся данные из справочника пользователей (`users`), показывать **Фамилию + инициалы** (например: «Иванов И.И.»), а не `__id` или `login`.

**Формат:** `<Фамилия> <Первая буква имени>.<Первая буква отчества>.`
- Если отчества нет — только `<Фамилия> <Первая буква имени>.`
- Если имени нет — `<Фамилия>`

**Исключение:** только по прямому запросу (например: «покажи логины», «выведи ID»).

**Поле в данных:** `fullname` — объект с ключами `lastname`, `firstname`, `middlename`.

**Пример преобразования:**
```python
def format_user(fullname):
    parts = [fullname.get('lastname', '')]
    if fullname.get('firstname'):
        parts.append(fullname['firstname'][0] + '.')
        if fullname.get('middlename'):
            parts[-1] += fullname['middlename'][0] + '.'
    return ' '.join(parts)
```
