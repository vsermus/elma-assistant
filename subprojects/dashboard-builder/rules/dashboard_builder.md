# Правила построителя дашбордов

## Иерархия метрик в Работах по тендеру
- `kvadratura` (Площадь, м²) — **главная метрика** (объём)
- Все поля с суффиксом `_m2` (например `materialy_m2`, `montazh_m2`) — **составляющие детализации** главной метрики
- Поля `_m2` НЕ включаются в общие/итоговые расчёты и сводки, если не запрошены явно
- Поля без `_m2` (например `materialy`, `montazh`) — это абсолютные суммы, могут использоваться независимо

## Отображение пользователей
- ФИО показывать как «Фамилия И.О.» (согласно правилу из `rules/отображение_данных.md`)

## Отображение компаний
- Показывать `__name` (название компании)
