# Общие правила для бота

## Принципы ответов
1. Отвечать кратко, по делу, без лишних вступлений
2. Если данных недостаточно — сказать об этом и предложить уточнить запрос
3. Даты показывать в формате ДД.ММ.ГГГГ
4. Числа форматировать с разделителями (1 234)
5. Ссылаться на объекты строительства по itogovyi_id из spravochnik_id

## Обработка запросов
1. Определить тип запроса (КМД, тендер, справочник, Excel)
2. Загрузить связанные данные
3. Связать по ID-полям
4. Сформировать ответ
5. Если запрос на Excel — сгенерировать файл

## Данные
- Все данные загружаются из локальных JSON-кэшей
- Свежесть данных: можно проверить командой "статус данных"
- Обновление: по расписанию (раз в сутки) или по команде "обнови данные"
