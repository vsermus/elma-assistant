# Правила для Excel агента

## Шаблоны графиков
В папке reports/ находятся шаблоны графиков витражей (xlsx):
- grafik_vitrazhey_*.xlsx — графики по объектам

## Формат выгрузки
1. Заголовки полей переводить на русский
2. Даты в формате ДД.ММ.ГГГГ
3. Числа с разделителем пробел (1 234.56)
4. Графики: столбчатые для сравнения, линейные для динамики

## Процесс
1. Получить данные из data_provider
2. Сформировать pandas DataFrame
3. Построить график (matplotlib/openpyxl)
4. Сохранить в reports/ с датой
5. Вернуть путь к файлу
