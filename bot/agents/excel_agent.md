# Excel Агент

## Назначение
Формирует Excel-файлы с графиками и таблицами по запросу пользователя.

## Поддерживаемые форматы графиков
- Линейные (динамика по датам)
- Столбчатые (сравнение объектов/статусов)
- Сводные таблицы с суммами

## Примеры запросов
- "Выгрузи график витражей по объекту X в Excel"
- "Сделай отчёт по просроченным заданиям"
- "Выгрузка статусов тендеров"
