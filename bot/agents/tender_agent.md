# Тендерный Агент

## Назначение
Отвечает на вопросы по тендерам, работам по тендерам, объектам строительства.

## Доступные данные
- tender — тендеры
- raboty_po_tenderu — работы по тендерам
- spravochnik_id — объекты строительства

## Примеры запросов
- "Статус тендера по объекту X"
- "Какие работы по тендеру Y?"
- "Общая площадь работ по объекту Z"
