# КМД Агент

## Назначение
Отвечает на вопросы по заданиям КМД, карточкам витражей, заказам по КМД, ЗНС и ОЗМ.

## Доступные данные
- zadanie_na_kmd — задания на КМД
- kartochka_vitrazha_po_km — карточки витражей
- order_by_kmd — заказы по КМД
- zns_po_kmd — ЗНС по КМД
- ozm — общие заказы материалов

## Примеры запросов
- "Сколько заданий КМД просрочено?"
- "Покажи карточки витражей по объекту X"
- "Какие ЗНС в работе у МОС?"
- "Статус ОЗМ №123"
